## 🚀 Getting started

When you open the application, you see the main workspace. This area displays your tracks and the navigation menu. 

### Importing audio
You can drag and drop your vocal files directly into the editor. The software supports formats like WAV, MP3, and FLAC. Once you import a file, the waveform appears in the center panel.

### Using the piano roll
The piano roll allows you to input notes for the synthesis engine. Click inside the grid to place notes. You can drag the corners of a note to change its length. This process controls the melody and pitch of the voice.

### Configuring the engine
Go to the settings menu to choose the vocal model. The application includes default models to get you started. If you have custom models, you can import them through the model manager folder. The engine uses ONNX runtime to process sounds, which ensures speed during playback.

## ⚙️ Voice conversion features

Voice conversion allows you to replace the voice in an existing track with a different model. Select the source audio file and click the convert button. The process takes several minutes depending on the length of your clip. You can change the pitch or character of the voice using the control sliders on the right panel.

## 🎛️ Advanced settings

Experienced users can adjust the processing buffers in the preferences menu. Large buffers improve stability on older hardware. Smaller buffers reduce latency when you play notes in real time. The software automatically detects your audio interface, but you can select specific hardware inputs if you have multiple microphones connected.

## 🤝 Getting help
